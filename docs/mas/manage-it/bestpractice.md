# MAS Manage IT

### Lab Result Highlights

- Based on lab tests, for sizing we recommend 50 users load per MAS Manage IT server bundle pod.

- Application Server UI Pod limits:
    - cpu 6 Cores
    - memory 16 GB

- DB2 limits:
    - cpu 14 Cores
    - memory 32 GB
