# MAS Manage IT

## Lab Result Highlights

- Based on lab tests, we recommend sizing 50 users load per MAS Manage IT server bundle pod.

- DB2 limits:
    - 100 users - 4 db2 cpu
    - 300 users - 8 db2 cpu
    - 500 users - 12 db2 cpu
    - 700 users - 16 db2 cpu
